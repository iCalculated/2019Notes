Electric Fields

Uniform fields are “nice” because calculus can be avoided.

Voltage is V=dE⃗V=d \vec{E}V=dE such that ddd is the direction along the field. Makes sense since voltage is potential.

“We don’t know about magnets”

Charge Density

Linear charge density will use λ\lambdaλ, charge per length. (dq=λdxdq=\lambda dxdq=λdx)
Surface charge will use σ\sigmaσ, charge per area. (dq=σdAdq = \sigma dAdq=σdA)
Volume charge will use ρ\rhoρ, char per volume. (dq=pdVdq=pdVdq=pdV)

Line of charge

Let length be LLL and total charge be QQQ, then λ=QL\lambda = \dfrac{Q}{L}λ=LQ​. We have E⃗=∫dE⃗=∫kdqr2r^P.\vec{E} = \int d\vec{E} = \int k \dfrac{dq}{r^2} \hat{r}_P.E=∫dE=∫kr2dq​r^P​.

Actually calculating turns ugly, unless if we exploit symmetry mercilessly. Logically, dq=λdxdq=\lambda dxdq=λdx, which can be shown with super hacky math:

$ dq = \lambda dx\implies \int dq = \int \lambda dx \implies Q = \lambda L \implies \lambda = \dfrac{Q}{L}.$

Let one end of the rod be at x=0x=0x=0 and the other at x=Lx=Lx=L, then we can rewrite the integral as

$ \vec{E} = \int_0^L k\dfrac{\lambda dx}{(d+L-x)^2}\hat{r}.$

Notably, direction is fake here, so

$ \vec{E} = \int_0^L k\dfrac{\lambda dx}{(d+L-x)^2}\hat{r} = \int_0^L k\dfrac{\lambda dx}{(d+L-x)^2}\hat{x}.$

Then u-sub, u=d+L−xu=d+L-xu=d+L−x which ultimately yields

λk∫d+Ld−duu2x^=λk[1d−1d+L]x^.\lambda k \int^d_{d+L} \dfrac{-du}{u^2}\hat{x} = \lambda k\left[\dfrac{1}{d}-\dfrac{1}{d+L}\right]\hat{x}.λk∫d+Ld​u2−du​x^=λk[d1​−d+L1​]x^.

Apply a trivial taylor expansion to rewrite the expression as

1d−1d+L=1d−1d(1+L/d)\dfrac{1}{d}-\dfrac{1}{d+L} = \dfrac{1}{d}-\dfrac{1}{d(1+L/d)}d1​−d+L1​=d1​−d(1+L/d)1​

and if we assume that d>>Ld>>Ld>>L for convenience, then L/d<<1L/d <<1L/d<<1. FOr any ∣δ∣<<1\vert\delta\vert << 1∣δ∣<<1, (1+δ)−1=1−δ(1+\delta)^{-1}=1-\delta(1+δ)−1=1−δ by physics hack magic, so the previous expression approaches

k(QL)Ld2=kQd2.k\left(\dfrac{Q}{L}\right)\dfrac{L}{d^2} = \dfrac{kQ}{d^2}.k(LQ​)d2L​=d2kQ​.

Oh look, it’s Coulomb’s Law, that looks rather compelling.
