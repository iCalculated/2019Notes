“Robots”

  • Allowed segments, revolute joints (with parameter θ\thetaθ), and prismatic joints (with parameter LLL, as a piston)
  • Questions:
    • In what configuration is the hand? (forward kinematic)
    • Where can the hand be? (also forward kinematics)
    • How to move between configuations? (reverse kinematics)
  • Create variables with respect to trig
  • Utilize limited mapping, for example f:S1×S1×[1,3]→A2×S1f: S^1\times S^1\times [1,3]\to \mathbb{A}^2\times S^1f:S1×S1×[1,3]→A2×S1
